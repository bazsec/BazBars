# BazBars Changelog

## 003
- Added AceDB profile support
  - Profile switching, creating, copying, and deleting
  - Per-profile bars and keybinds
  - Copy profiles between characters

## 002
- Initial public release
- Custom extra action bars independent of Blizzard's action bar system
- Drag spells, items, macros, mounts, battle pets, equipment sets onto buttons
- Full Edit Mode integration with grid snapping
- Blizzard-native button appearance
- Visibility macros (combat, target, mouseover, custom)
- Quick Keybind Mode
- Masque skinning support
- Minimap button
- Mouseover fade and bar opacity
