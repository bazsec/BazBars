# BazBars Changelog

## 008 - Power User Update
### New Features
- **Duplicate Bar** - Clone any bar with all its button assignments via Edit Mode or `/bb duplicate <id>`
- **Right-Click Self-Cast** - Per-bar option to cast spells/use items on yourself with right-click
- **Macrotext Editor** - Write custom macro conditionals for any button (Edit Mode > Behavior > Edit Button Macrotext)
- **#showtooltip Support** - Buttons with custom macrotext display the icon and tooltip of the spell in your `#showtooltip` line
- **Import/Export Bar Configs** - Share your bar setups as copyable strings (Edit Mode > Actions > Export Bar Config, or `/bb export <id>`)

### Improvements
- Collapsible settings panel sections (Layout, Appearance, Behavior, Actions)
- Inline nudge controls
- Show Minimap Button toggle in BazBars Settings
